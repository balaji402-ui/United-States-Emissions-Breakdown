# 🌍 Emissions Dashboard using Databricks SQL

## Project Overview
This project analyzes greenhouse gas emissions using Databricks SQL and interactive dashboards. It provides insights into emissions by state and county and supports data-driven environmental analysis.

## Tools Used
- Databricks SQL
- Databricks Dashboards
- SQL
- GitHub

## Dataset
Emissions 2023

## Dashboard Features
- Total Emissions
- Top 10 States
- Top Counties
- Location Analysis
- Interactive Filters
